Estudando HTML


Nossos estudam acabam falando a respeito de um item muito sério que é organização. Isso porque a linguagem HTML é responsavel pela marcação dos elesmentos da janela do navegador. O HTML não cria nada, ele manda buscar ou ultiliza as letrinhas que você digita.


Páragrafos



Para colocarmos parágrafos colocamos os elementos "p" e "/p". Para manter uma quebra de linha, com o mesmo parágrafo, usamos o elemento "br". Temos ainda parágrafos essesnciais, denominados títulos, cabeçalhos ou headings que são usados elementos de "h1" a "h6" e seus repectivos fechamentos de "/h1" a "/h6".


imagens



Uma imagem aparece na janela do navegador pelo uso elemento do elemento "img". Esse elemento possui o atributo "src", responsavel pela localização da imagem. Para trabalhar melhor com a imagem, você pode coloca-la dentro das tags de marcação dos parágrafos.
